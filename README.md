## Requirements

[Python3](https://www.python.org/downloads/windows/)\
[Node.js](https://nodejs.org/en)

## Setup

`git clone https://github.com/dharunvs/octal-encryption-website.git \
cd octal-encryption-website`

> Open terminal #1\
> `cd backend`\
> `pip install Pillow`\
> `npm install`\
> `npm run dev`

> Open terminal #2\
> `cd frontend`\
> `npm install`\
> `npm run dev`
